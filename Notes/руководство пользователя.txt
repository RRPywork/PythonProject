Информационно-аналитическое приложение "Процессоры"

Назначение приложения - учебное приложение для анализа различных характеристик процессоров

Системные требования:
Архитектура: x86-64
Операционная система: windows 7
Процессор intel core i3 3240
Графика: intel hd graphics 4000
Оперативная память: 4Гб
Место на жестком диске: 1Гб

Инструкция по установке - распаковать архив в желаемое место

Инструкция по настройке - изменить файл config.ini по необходимости. Можно изменить разрешение экрана и пути к основным справочникам

Инструкция по использованию:

Приложение запускается путем исполнения файла Scripts\main.py
Приложение состоит из двух вкладок. Первая открывается при запуске и представляет отображение 
сессионного справочника (на основе всех данных основных справочников). 
Сессионный справочник (СС) - единственный, который изменяется на протяжении работы приложения.
Над сессионным справочником допустимы следующие операции:
  1. Изменение значения ячейки
  2. Добавление/удажение объекта
  3. Переименование объекта
  4. Отображение (СС может отображать часть данных, содержащтхся в основных справочниках)
  5. Сохранение и загрузка сессии - СС можно сохранить в файл.
Для изменения основных справочников можно сохранить СС в файл, с именем основного справочника

Вторая вкладка предназначена для осуществления анализа данных СС.
Для формирования отчета необходимо выбрать тип отчета и атрибуты, по которм строится отчет.ъ
Отчеты можно сохранять в файлы.
